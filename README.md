# yTax

This repository is a public feedback channel for **yTax**. It intentionally contains no source code — just this README and the repository's license.

## What this repo is for

We use GitHub Issues here to collect feedback, feature requests, and bug reports from early users and anyone else interested in yTax. If there's something you'd like to see, or something that isn't behaving the way you'd expect, please open an issue.

## How to submit feedback

1. Search open and closed issues first, to avoid duplicates.
2. Open one issue per idea, bug, or piece of feedback.
3. Include specifics — what you expected, what happened instead, and any relevant context (screenshots, links, examples).

We read every issue, but can't guarantee a response to each one.
